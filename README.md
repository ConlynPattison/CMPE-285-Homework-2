# CMPE-285-Homework-2

Description:
- Reads finance input (e.g., stock symbols, date range) from config/CLI (see `main.py`).
- Outputs results to console and/or files (summary stats, returns, etc.).

Install dependencies in venv:
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

To run:
```bash
source .venv/bin/activate
python main.py
```
